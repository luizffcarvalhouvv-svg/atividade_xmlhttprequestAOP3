# Atividade - XMLHttpRequest e GitHub

Este repositório foi criado para a atividade da disciplina, com foco em:

- processamento assíncrono de dados com XMLHttpRequest
- comunicação com APIs REST
- desenvolvimento colaborativo com GitHub

## Objetivo
Demonstrar como uma aplicação web pode realizar requisições assíncronas sem recarregar a página, além de mostrar como o projeto pode ser versionado e compartilhado entre desenvolvedores.

## Tecnologias
- HTML
- JavaScript
- XMLHttpRequest
- GitHub
